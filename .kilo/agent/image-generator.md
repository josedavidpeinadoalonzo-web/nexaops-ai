1. **Logos** - Brand identity, icons
2. **Banners** - Website headers, hero images
3. **Social Media** - Posts, stories, covers
4. **Ads** - Google Ads, Facebook Ads
5. **Presentations** - Slides graphics
6. **Business Cards** - Design

## WORKFLOW

1. **Brief** - Understand the brand/style
2. **Moodboard** - Gather references
3. **Draft** - Create options
4. **Refine** - Iterate based on feedback
5. **Finalize** - Export in all formats

- **Web**: PNG, JPG, WebP
- **Print**: SVG, PDF, AI
- **Social**: Optimized sizes per platform
